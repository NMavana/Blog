# Contributing to my PHP Blog 

First off, thanks you for considering contibuting to this project! It's people like you who make the open-source community such an amazing place to learn and create.


## How Can You Help?
I am Specifically looking for help to make the design perfect.You can contibute by:
- Improving the UI: Ensuring the blog's layout using Bootstrap or custom CSS.
- Responsive Design: Ensuring the blog looks great on mobile, tablet, and Desktop screens.
- Frontend Logic: Adding JavaScript for smoother user interaction.

## Style Guidelines
- CSS/BOOTSTRAP: Please keep the styling consistent with the exixting theme. Ensure that all custom CSS must be in another file like `customstyle.css`
- PHP/HTML: Ensure HTML is semantic and properly nested within PHP tags.
- Comments: Add comments to your code so I can understand the design changes you've made.

## Your Contribution Process 
1. Fork the `feature/AmazingDesign` branch: Create your own copy of the repository. _Ensure that the `main` branch is unchecked_
2. Switch Branch Locally: Once you have forked the repo to your account, clone it to your computer and switch to that specific feature branch:
`git checkout feature/AmazingDesign`
3. Commit Your Change: Provide a clear message, e.g `git commit -m "Add modern login Style"`.
4. Push to the branch: `git push origin feature/AmazingDesign`. 
5. Open a Pull Request: Describe your design changes and include screenshots if possible.

## Testing Your Changes
Before submitting a Pull Request, please ensure:
- 
- The database connection in `connection.php` still works.
- The CRUD operation (Create, Read, Update, Delete) are not broken by the new design.
